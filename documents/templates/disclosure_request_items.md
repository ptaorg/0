# 公文書開示請求項目例

対象年度：＿＿年度から＿＿年度
対象校：＿＿＿＿＿＿学校

1. PTA入会案内、PTA入会申込書、退会届、PTA会則、PTA個人情報取扱規程その他PTA加入・退会・会員管理に関する文書
2. 学校徴収金案内、口座振替依頼書、PTA会費を含む徴収案内、学校徴収金とPTA会費の区分が分かる文書
3. 学校アプリ・学校メール・学校配布物を用いてPTA文書を配信・配布することに関する規約、仕様書、運用要領、保護者説明文、内部決裁文書
4. 学校が保有する児童・保護者情報をPTAへ提供し、またはPTA目的で利用することに関する同意書、説明文、内部決裁文書、教育委員会通知
5. 教職員がPTA業務を行うことに関する校務分掌、職務専念義務免除、兼職兼業、服務上の整理、教育委員会と学校との打合せ記録
6. 教育委員会が学校に発出したPTA、学校徴収金、個人情報、学校関係団体、教職員のPTA関与に関する通知、依頼、事務連絡
