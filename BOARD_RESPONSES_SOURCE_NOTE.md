# 教育委員会の回答集ページについて

このページは、以前作成済みだった `board-responses.html`（地図付き回答集）と、教育委員会回答整理データの構成を、現在のTemplate-Party系サイトデザインに合わせて復元したものです。

## 反映内容

- `board-responses.html` を新規追加
- Leaflet による自治体マップを追加
- 横浜市通知、厚木市、磐田市、大阪府、町田市、高根沢町などを抜粋カード化
- `support.html` を新規追加
- PayPay支援リンクを `https://qr.paypay.ne.jp/p2p01_7qyY46XqZxGgVsQw` に設定

## 公開前確認

各自治体回答の要約は、旧ページ・整理データを元にした復元です。公開確定前に、PDF原本・メール本文・公文書開示資料との照合を行ってください。
