# PTA適正化推進委員会 GitHub Pages版 v4

Template-Party系の学校サイト風テンプレートをベースに、PTA適正化推進委員会の静的サイトとして再構成したものです。

## 主な内容

- `index.html`：公開向けトップページ
- `guide.html`：PTA運営適正化ガイド入口
- `guidebook.html`：公立学校におけるPTA運営の適正化に関するガイドブック第4版
- `laws.html`：法令・一次資料リンク集
- `documents.html`：資料室、テンプレート、公文書開示請求項目例
- `parents.html` / `officers.html` / `schools.html` / `boards.html`：立場別入口

## GitHub Pagesでの公開

1. このZIPの中身をリポジトリ直下へ配置
2. GitHub Pagesの公開ブランチを `main` / root に設定
3. `index.html` がトップページになります

## 注意

- このサイトは学校公式サイトではありません。
- Template-Partyの著作権表示は残しています。
- 外部リンクは公開前に再確認してください。
- 実際のPDF資料を追加する場合は、個人情報・印影・QRコード・有効期限付きURLを確認してください。
